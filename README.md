<span align="center">Creating a simple API using Node.js and PostgreSQL</span>

## Getting Started

To run the project, you need to have Node (and NPM) installed:

```bash
npm install
```
or
```bash
yarn install
```

### Usage

**URL:** localhost:5000

```
[GET] /users
[GET] /users/:id
[POST] /users
[PUT] /users/:id
[DELETE] /users/:id
```
