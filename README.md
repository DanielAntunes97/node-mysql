>Creating a simple API using Node.js and PostgreSQL

## Installation

To run the project, you need to have Node (and NPM) installed:

```console
foo@bar:~$ npm install
```
or
```console
foo@bar:~$ npm install
```

### Usage

localhost:5000

```
[GET] /users
[GET] /users/:id
[POST] /users
[PUT] /users/:id
[DELETE] /users/:id
```
